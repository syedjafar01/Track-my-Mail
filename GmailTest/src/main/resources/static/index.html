TrackMyMail is a web based application which will return the status of a each conversation between a client and developer, provided the TrackMyMail Email ID is kept as CC(Carbon Copy) by both the parties.
The TrackMyMail shows the number of conversations between various Clients and Developers and shows the number of replied and not replied mails.The Application also would alert the administrator if the number of not replied mails goes up alarmingly.
This Application is designed for high-priority clients of Big Brain Project who might have queries which has to replied without fail,soon as possible,by the developers. This Application requires an administrator who would have access to the application who would put it on a monitor helping developers to understand the number of mails which are not replied back to clients.
